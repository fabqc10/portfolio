Infinity Scroll

This project implements infity scroll functionality. This is something that happens on social media as the user scroll.

As the users scrolls near the bottom of the page a network request is made but it happens before the user reach the bottom and it feels like the user is scrolling forever.

The images are photos fetch from the Unsplash API. If the user click on the image is able to see that image on the splash website and see the author and download the photo.
